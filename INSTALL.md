# OneMCP Installation Guide

## Quick Install (Linux/macOS)
curl -fsSL https://raw.githubusercontent.com/mdarshad-ai/OneMCP/main/install.sh | bash

## Manual Install
1. Download the appropriate binary for your platform
2. Make it executable: chmod +x onemcp
3. Move to PATH: sudo mv onemcp /usr/local/bin/

## Getting Started
onemcp add filesystem @modelcontextprotocol/server-filesystem
onemcp start
